# Ecommerce-Sales-Analysis


## Project Overview

This project analyzes e-commerce sales data using Python. It demonstrates the complete data analysis workflow including data loading, cleaning, exploratory data analysis (EDA), visualization, and business insights.

## Objectives

* Load and preprocess sales data.
* Analyze sales performance by product and region.
* Visualize trends using different chart types.
* Generate meaningful business insights.
* Demonstrate a complete data analysis pipeline.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

## Dataset

The dataset contains the following columns:

* Date
* Product
* Quantity
* Price
* Customer_ID
* Region
* Total_Sales

## Setup Instructions

1. Install Python 3.10 or above.
2. Install Jupyter Notebook or Anaconda.
3. Clone or download this repository.
4. Install the required libraries:

pip install -r requirements.txt

5. Open analysis.ipynb in Jupyter Notebook.
6. Run all notebook cells from top to bottom.
